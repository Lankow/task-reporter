In order to make application work properly:
1.In text file "observedProcesses.txt" - Put process name as it appears in the window - Its case sensitive so be sure to type it the same
2.Launch "Launch task-reporter"
OR 
2.Run cmd.exe
3.In cmd.exe go to task-reporter directory in order to do so type (XXXX stand for the rest of the path):
cd C:\XXXX\task-reporter
4.When you are in the proper directory type in cmd.exe:
java -jar task-reporter.jar
5.Enjoy usage of task reporter - In order to save your report press ENTER
